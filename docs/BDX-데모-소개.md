# BDX — Beauty Decision eXperience

> 네일샵 현장 상담 & 고객 관리 플랫폼 데모

**데모 URL**: https://bdx-demo.vercel.app

---

## BDX가 뭔가요?

BDX는 네일샵에서 **고객 상담부터 시술 확인까지** 한 번에 처리할 수 있는 태블릿/모바일 기반 플랫폼입니다.

기존에 종이나 카카오톡으로 주고받던 상담 내용을 **앱 하나로 체계적으로 관리**할 수 있습니다.

---

## 주요 기능

### 1. 상담 플로우 (5단계)

고객이 매장에 오면, 디자이너가 태블릿으로 아래 순서대로 상담을 진행합니다.

| 단계 | 화면 | 하는 일 |
|------|------|---------|
| **1단계** | 고객 정보 | 고객 이름, 연락처, 언어 선택 (외국인 고객 대응) |
| **2단계** | 시술 범위 | 손/발, 한손/양손 선택 |
| **3단계** | 디자인 선택 | 원컬러, 그라데이션, 프렌치 등 시술 타입 선택 |
| **4단계** | 표현 & 파츠 | 매트, 글리터, 스톤 등 세부 표현 선택 |
| **5단계** | 네일 캔버스 | 손가락별로 어떤 시술을 할지 직접 지정 + 메모 입력 |

상담이 끝나면 **시술 확인서**가 자동으로 생성됩니다.

### 2. 시술 확인서

상담 완료 후 바로 보여지는 확인서입니다.

- **시술 체크리스트**: 오늘 시술할 내용을 항목별로 체크
- **네일 디자인 탭**: 손가락별 시술 내용과 메모를 한눈에 확인
- **참고 이미지**: 상담 중 첨부한 레퍼런스 이미지 표시
- **가격 & 소요시간**: 자동 계산된 예상 금액과 시간

고객에게 보여주며 "이렇게 시술할게요"라고 확인받는 용도입니다.

### 3. 홈 대시보드

디자이너가 로그인하면 보이는 메인 화면입니다.

- **오늘의 예약**: 당일 예약 목록 한눈에 확인
- **최근 상담**: 최근 진행한 상담 기록 빠른 접근
- **새 상담 시작**: 원터치로 상담 시작
- **빠른 통계**: 오늘/이번 주 상담 건수, 매출 요약

### 4. 기록 관리

과거 상담/예약 기록을 3가지 뷰로 관리합니다.

- **주간 타임그리드**: 일주일 단위로 예약/상담을 시간대별로 확인
- **월간 캘린더**: 한 달 전체 일정을 한눈에 파악
- **리스트 뷰**: 전체 기록을 목록으로 검색/필터링

예약 카드를 탭하면 상세 정보 확인 및 수정이 가능하고, 참고 이미지 첨부도 지원합니다.

### 5. 고객 관리

단골 고객의 정보를 체계적으로 관리합니다.

- **기본 정보**: 이름, 연락처, 선호 언어
- **시술 이력**: 과거 상담/시술 기록 전체 조회
- **스몰토크 메모**: 상담 중 나눈 대화 메모 (예: "다음 달 결혼식", "알러지 있음")
- **담당 디자이너**: 고객별 담당 디자이너 지정

### 6. 설정

매장 운영에 필요한 설정을 관리합니다.

- **매장 정보**: 상호명, 주소, 영업시간
- **시술 메뉴 & 가격**: 시술 종류별 기본가, 추가금 설정
- **디자이너 관리**: 소속 디자이너 등록/관리
- **테마 커스터마이징**: 매장 브랜드에 맞는 앱 테마 설정
- **언어 설정**: 한국어/영어/일본어/중국어 지원

## 핵심 특징

### 다국어 지원
외국인 고객이 많은 매장을 위해 **한국어, 영어, 일본어, 중국어**를 지원합니다. 상담 시작 시 고객 언어를 선택하면 상담 화면이 해당 언어로 전환됩니다.

### 자동 가격 계산
시술 범위, 디자인 타입, 추가 표현(파츠, 글리터 등)을 선택하면 **가격이 자동으로 계산**됩니다. 할인 적용도 가능합니다.

### 손가락별 개별 시술 지정
네일 캔버스에서 **각 손가락마다 다른 시술**을 지정할 수 있습니다. "엄지는 풀아트, 검지는 원컬러" 같은 복잡한 주문도 깔끔하게 기록됩니다.

### 디자이너별 색상 구분
타임그리드에서 디자이너마다 **다른 색상**으로 예약이 표시되어, 누가 언제 어떤 시술을 하는지 한눈에 파악할 수 있습니다.

## 데모 사용 방법

1. https://bdx-demo.vercel.app 접속
2. 스플래시 → 인트로 → 로그인 화면 순서로 진행
3. **PIN 입력**: `1234` 입력하여 로그인
4. 홈 화면에서 "새 상담 시작" 버튼으로 상담 플로우 체험
5. 하단 탭으로 기록/고객/설정 화면 이동

> 데모 데이터가 미리 세팅되어 있어 바로 체험할 수 있습니다.

## 지원 기기

- **태블릿** (iPad 등): 매장 내 상담용 최적화
- **모바일**: 이동 중 간단한 확인용
- **데스크톱**: 관리자용 넓은 화면 지원

반응형 디자인으로 모든 화면 크기에서 사용 가능합니다.
