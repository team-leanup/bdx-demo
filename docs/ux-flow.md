# BDX 사용 가이드 — 네일샵 사장님용

BDX는 네일샵 사장님이 **스케줄 관리 + 고객 관리 + 매출 기록**을 하나로 관리하는 앱입니다.

---

## 핵심 개념

| 개념 | 뜻 | 예시 |
|------|------|------|
| **예약** | 고객이 올 시간을 잡는 것 | "3/27 14:00 김미소님 젤네일" |
| **고객** | 우리 가게에 온 사람 | 이름, 전화번호, 선호 스타일, 특이사항 |
| **시술 기록** | 시술 내용 + 결제 금액 | "젤 원컬러 55,000원" |
| **매출** | 사장님이 확인한 최종 금액 | "결제 완료" 버튼 누르면 매출로 잡힘 |

### 데이터 관계

```
예약 등록 ──→ 고객 자동 등록 (새 고객이면)
   │
   ▼
시술 진행
   │
   ▼
매출 등록 ──→ 시술 기록 자동 생성
   │           └──→ 고객 총매출/방문횟수 자동 갱신
   ▼
결제 완료 ──→ "오늘 매출"에 반영
```

**쉽게 말하면:**
- 예약하면 → 고객이 자동으로 생겨요
- 매출 등록하면 → 시술 기록이 자동으로 생겨요
- 결제 완료 누르면 → 오늘 매출에 잡혀요

---

## 사장님의 하루 (기본 플로우)

### 1. 아침 — 오늘 스케줄 확인

홈 화면에서:
- **오늘 스케줄**: 오늘 올 고객 목록
- 각 고객 카드에서 바로 **고객 정보** 확인 가능
- 기존 고객이면 → [고객 카드] + [매출 등록]
- 처음 오는 고객이면 → [상담 시작] + [매출 등록]

### 2. 고객 도착 — 시술

**단골 고객** (대부분의 경우):
1. 예약 카드 터치 → 인라인 프리뷰로 특이사항 확인
2. 바로 시술 진행
3. 시술 끝나면 → **"매출 등록"** 클릭

**처음 오는 고객 / 외국인**:
1. 예약 카드에서 → **"상담 시작"** 클릭
2. 다국어 상담 플로우 진행 (디자인 선택, 특성 확인 등)
3. 상담 완료 → 시술 → **"결제 완료"**

### 3. 매출 등록

두 가지 방법:
- **예약에서**: 예약 카드 → "매출 등록" 버튼
- **직접**: 홈 → "매출 등록" 버튼 (예약 없이 바로)

매출 등록하면:
- 시술 기록 자동 생성
- 고객이 없으면 자동 생성
- 결제 완료 즉시 처리 (퀵세일)

### 4. 결제 완료

- **퀵세일(매출 등록)**: 등록하면 바로 결제 완료
- **상담 후**: 시술 기록 상세에서 **"결제 완료"** 버튼을 눌러야 매출에 잡힘
- 결제 전까지는 "결제 전" 뱃지가 표시됨

## 화면별 안내

### 홈 화면

| 영역 | 기능 |
|------|------|
| 오늘 스케줄 | 오늘 예약된 고객 리스트 |
| 새 예약 등록 | 새 예약 추가 |
| 매출 등록 | 예약 없이 바로 매출 기록 |
| 신규 고객 상담 | 처음 오는 분/외국인용 상담 |
| 오늘 매출 | 결제 완료된 금액 합계 |

### 기록 탭 (스케줄 + 시술 기록)

**스케줄 탭:**
- 주간 타임그리드 → 디자이너별 일정 한눈에
- 예약 터치 → 바텀시트로 상세 정보 + 액션 버튼

**시술 기록 탭:**
- 지금까지 한 시술 전체 목록
- 이름/전화번호/서비스명으로 검색
- 각 기록에서 결제 상태 확인 (결제 전/완료)

### 고객 탭

- 전체 / 단골 / 일반 필터
- **"새 고객 등록"** → 간단 폼 (이름, 전화, 메모)
- 고객 카드: 총매출, 방문횟수, 태그, 시술 이력, 메모

### 대시보드

- 오늘/이번 달 매출
- 인기 시술 종류
- 예약 건수 통계

## 고객 등록 방법 (3가지)

| 방법 | 언제 쓰는지 |
|------|-----------|
| 예약 등록 | 전화/카톡으로 예약받을 때 → 자동 등록 |
| 매출 등록 | 시술 끝나고 결제할 때 → 자동 등록 |
| 고객 탭에서 직접 | 미리 고객 정보만 넣어두고 싶을 때 |

어떤 방법이든 **같은 전화번호면 중복 생성 안 됨** — 기존 고객에 자동 연결

---

## 상담 기능 (보조 기능)

상담은 **처음 오는 고객**이나 **외국인 고객**에게만 필요합니다.

- 4개 언어 지원 (한국어, 영어, 중국어, 일본어)
- 디자인 선택 → 특성 확인 → 요약 → 시술
- 단골 고객은 상담 없이 바로 시술 + 매출 등록

**상담 단계:**
1. 고객 정보
2. 시술 기본 (손/발, 범위)
3. 디자인 선택
4. 특성 (표현, 캔버스)
5. 최종 확인 + 저장

## 결제 흐름 정리

```
[상담 완료]                    [퀵세일/매출 등록]
    │                              │
    ▼                              ▼
시술 기록 생성                시술 기록 생성
(결제 전 상태)                (결제 완료 상태)
    │                              │
    ▼                              ▼
"결제 완료" 버튼              바로 매출에 반영
    │
    ▼
매출에 반영
+ 고객 총매출 갱신
+ 방문횟수 +1
```

## 자주 쓰는 동작 요약

| 하고 싶은 것 | 방법 |
|-------------|------|
| 예약 잡기 | 홈 → "새 예약 등록" |
| 매출 기록 | 홈 → "매출 등록" 또는 예약 카드 → "매출 등록" |
| 고객 등록 | 고객 탭 → "새 고객 등록" |
| 외국인 상담 | 홈 → "신규 고객 상담" |
| 오늘 매출 확인 | 홈 하단 "오늘 매출" 카드 |
| 고객 특이사항 확인 | 예약 카드 터치 → 인라인 프리뷰 |
| 과거 기록 검색 | 기록 탭 → 시술 기록 → 검색 |
