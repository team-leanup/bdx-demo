[BDX '포트폴리오' 기능 개선사항 (MVP 개발용)]

1. 등록 프로세스: "타이핑 제로" 전략
원장님이 사진을 찍어 올릴 때, 다음 정보들이 자동으로 매칭되어야 합니다.

예약 데이터 연동: 어떤 손님인지 선택하면, 그 예약에 기록된 
[시술 가격], [손/발 구분], [젤 제거 여부]가 자동으로 입력창에 채워짐. (원장님은 확인만)

재료 퀵 태그 (Chip UI): 텍스트 입력창 대신, 하단에 [지아미 베이스] [빌더젤] [인기핑크#01] 같은 버튼을 배치해 툭툭 터치해서 입력

지난 레시피 복사: "이 손님 지난번이랑 똑같이 했어요" 버튼 클릭 시 지난번 재료 데이터 그대로 복사


2. 필터 시스템: "1초 만에 디자인 찾기"
단순 날짜순이 아니라, 손님 앞에서 바로 보여줄 수 있는 상황별 필터를 상단에 배치합니다.

분위기 필터: #심플, #화려, #웨딩, #키치/Y2K, #글리터
가격대 필터: 5만원 이하, 5~8만원, 8~12만원, 12만원 이상
글로벌 베스트: 외국인 고객들이 유독 많이 찾는 디자인만 따로 모아보기. (K-뷰티 엣지)


3. 상세 보기: "시각적 레시피"

사진만 달랑 보여주는 게 아니라, 사진을 누르면 정보가 사진 위에 겹쳐서(Overlay) 보여야 합니다.

정보 레이어: 사진 구석에 Price: 85,000, Time: 90min, Color: #자석핑크 정보를 반투명하게 노출

상담 연동: 사진 하단에 [이 디자인으로 상담지 보내기] 버튼 배치 (다음 예약 고객에게 예시로 보내기 위함)


4. 인스타 연동

외부 레퍼런스를 가져오는 기능 대신, 내 작업을 밖으로 내보내는 기능에 집중합니다.

해시태그 자동 생성: 입력된 재료와 스타일 태그를 조합해 #성수네일 #자석네일 #BDX 등 인스타용 해시태그를 자동으로 만들어주는 [텍스트 복사] 버튼 제공.

외부 레퍼런스 제외: 포트폴리오 탭에서는 '남의 사진'을 빼고 오직 원장님의 시술 자산만 관리하여 데이터 순도를 높임
