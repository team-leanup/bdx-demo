### 🎨 오후: 전체 플로우 QA + 메뉴판 대개편

#### 1. **손님 초대 링크** (신규 핵심 기능)

지금까지는 손님이 온 뒤 현장에서 상담했는데, 이제는 **방문 전에** 링크 하나만 보내면 고객이 스스로:

- 날짜와 시간을 고르고
- 손/발 중 시술 부위 선택
- 디자인 카테고리(심플/프렌치/자석/아트)
- 네일 상태, 컬러 선호, 특이사항까지 입력

그러면 매장에 도착할 때쯤 이미 상담이 다 끝난 상태가 됩니다. 외국인 고객도 한/영/중/일 중 본인 언어로 직접 작성 가능.

**사장님 화면**: 홈에 "손님 초대 링크 카드" 추가 → 기간·디자이너·시술시간·슬롯간격을 설정하면 영업시간과 기존 예약을 고려해 **예약 가능한 시간대를 자동 계산**해줍니다.

#### 2. **포트폴리오 → 메뉴판으로 재탄생**

기존의 "사진 모음" 느낌이었던 포트폴리오 화면을 **진짜 메뉴판**처럼 탈바꿈:

- 사진 썸네일이 **1.5배 커지고** 옆에 메뉴 이름·가격이 정갈하게 배치
- 카테고리별로 그룹핑 ("심플/원컬러 2개", "프렌치 1개", ...)
- **평상시에는 깔끔한 메뉴판**만 보이고, "수정" 버튼을 누르면 편집 모드 진입
  - 편집 모드에서만 연필(이름 수정), 해제, 추천·인기 뱃지 토글, 카테고리 이동 버튼이 나타남
  - 이전에는 연필 아이콘이 항상 보여서 복잡했던 걸 개선

#### 3. **대시보드 자동 조언 (AI처럼 친절한 코멘트)**

이전: "전월 대비 상담 건수 **-52%**" (빨간색) — 너무 강력한 느낌이고 무엇을 해야 할지 몰랐음

개선:

- 빨간색을 부드러운 장미색(rose)으로 톤 다운
- "-52%" 대신 "↘ 52%" 화살표 표기로 부정 기호의 강도 완화
- **상황에 맞는 조언 자동 표시**:
  - +30% 이상: "매출이 크게 늘었어요. 상승 흐름을 이어가보세요."
  - +10~30%: "꾸준히 성장 중이에요."
  - -10%~-30%: "감소 폭이 커지고 있어요. 단골 고객 연락을 권장해요."
  - -30% 이하: "매출이 많이 줄었어요. 골든타임 대상에게 리마인더를 보내보세요."
- 시간대별 예약 차트에도 "N시가 가장 바쁩니다. 보조 인력 배치를 고려해보세요" 같은 팁

#### 4. **고객 목록 깔끔하게**

- 아바타 옆에 떠 있던 주황색/빨간색 ⚠ 경고 뱃지 제거 → 고객 목록이 훨씬 덜 자극적으로 보임 (안전 태그는 상세 페이지에서 확인 가능)

#### 5. **사전상담 첫 화면에 "손/발 선택" 먼저**

- 네일이 **손**인지 **발**인지 먼저 토글로 선택한 뒤 디자인을 고르도록 순서 변경
- 이후 모든 단계에서 이 정보가 유지되고 확인 페이지에도 "🖐 손" 혹은 "🦶 발"이 표시됨
