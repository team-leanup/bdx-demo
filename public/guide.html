01

BDX가 뭔가요?

BDX는 네일샵 사장님이 매일 쓰는 관리 앱이에요.

하나의 앱에서 이 세 가지를 한꺼번에 할 수 있어요:

  • 스케줄 관리 — 오늘 올 고객, 예약 일정
  • 고객 관리 — 이름, 선호 스타일, 특이사항 기록
  • 매출 기록 — 시술 금액, 오늘/이번 달 매출 확인
외국인 고객이 왔을 때는 다국어 상담 기능으로 영어·중국어·일본어로 소통할 수 있어요.
02

핵심 개념 4가지

예약
고객이 올 시간을 잡는 거예요. 예약하면 고객도 자동으로 등록돼요.
고객
우리 가게에 온 사람이에요. 이름, 전화번호, 선호 스타일을 기록해요.
시술 기록
어떤 시술을 했는지, 얼마에 했는지 남기는 기록이에요.
매출
"결제 완료"를 누르면 오늘 매출에 잡혀요. 내가 확인해야 반영돼요.

데이터가 자동으로 연결돼요

1
예약 등록고객이 처음이면 자동으로 고객 등록까지
2
시술 진행예약된 시간에 고객이 와서 시술
3
매출 등록시술 끝나면 금액 입력 → 시술 기록 자동 생성
✓
결제 완료"결제 완료" 누르면 오늘 매출에 반영
03

사장님의 하루

아침 — 오늘 스케줄 확인

앱을 열면 "오늘 스케줄"에 오늘 올 고객이 쭉 나와요.

각 고객 카드를 터치하면 특이사항, 최근 시술, 메모를 바로 볼 수 있어요.

고객 도착 — 시술

단골 고객

대부분 이 경우예요.

  1. 예약 카드에서 특이사항 확인
  2. 바로 시술
  3. 끝나면 "매출 등록" 터치

신규 / 외국인

처음 오거나 외국인일 때.

  1. 외국인 → "상담 시작" 터치 / 한국인 신규 → "고객 등록 + 상담" 터치
  2. 다국어 화면으로 디자인 선택
  3. 상담 끝나면 시술
  4. 끝나면 "결제 완료"

시술 후 — 매출 등록

두 가지 방법이 있어요:

  • 예약 카드 → "매출 등록" 버튼 (자주 쓰는 방법)
  • 홈 화면 → "매출 등록" 큰 버튼 (예약 없이 바로)
매출 등록하면 시술 기록이 자동으로 만들어지고, 고객이 없으면 자동 등록까지 돼요.

퇴근 전 — 매출 확인

홈 하단 "오늘 매출"에서 오늘 총 매출을 확인하세요.

대시보드에서 이번 달 매출, 인기 시술 종류도 볼 수 있어요.

04

화면별 사용법

홈 화면

앱을 열면 가장 먼저 보이는 화면이에요.

영역설명
오늘 스케줄오늘 예약된 고객 리스트
새 예약 등록새 예약을 잡아요
매출 등록예약 없이 바로 매출 기록
신규 고객 상담처음 오는 분 · 외국인용
오늘 매출결제 완료된 금액 합계

기록 탭

상단에 "스케줄"과 "시술 기록" 두 개의 탭이 있어요.

스케줄 탭 — 주간 타임그리드로 디자이너별 일정을 한눈에 볼 수 있어요.

  • 빈 시간대를 꾹 누르면 바로 예약 등록
  • 예약을 터치하면 상세 정보 + 액션 버튼

시술 기록 탭 — 지금까지 한 시술 전체 목록이에요.

  • 이름, 전화번호, 서비스명으로 검색 가능
  • 결제 전/완료 상태를 한눈에 확인

고객 탭

  • 전체 / 단골 / 일반 필터로 나눠서 볼 수 있어요
  • "새 고객 등록"으로 이름 + 전화 + 메모만 넣으면 끝
  • 고객 카드에서 총매출, 방문횟수, 태그, 시술 이력, 메모 확인

대시보드

매장 현황을 숫자로 보여줘요 — 오늘/이번 달 매출, 인기 시술, 디자이너별 실적.

현장모드 탭

예약 카드에서 "시술 시작"을 탭하면 현장모드가 시작돼요.

  • 디자인 카테고리 선택 → 포트폴리오에서 사진 선택 또는 새로 촬영
  • 시술 타이머 + 추가 옵션 관리
  • 정산 후 Wrap-up 화면에서 공유카드 생성

포트폴리오 탭

시술 작품 사진을 모아두는 공간이에요. Wrap-up에서 저장한 사진이 자동으로 쌓여요.

  • 스타일 카테고리 태그 (심플/프렌치/자석/아트)로 분류
  • 상담 시 포트폴리오에서 디자인을 직접 골라 보여줄 수 있어요
경로: 설정 → 포트폴리오 탭

설정

매장 정보, 가격, 디자이너 관리 (추가/수정/PIN 변경), 테마, 영업시간 등.

06

고객 등록 방법

세 가지 방법이 있어요. 같은 전화번호면 중복 등록이 안 돼요 — 자동으로 기존 고객에 연결돼요.

1
예약할 때 자동 등록예약 등록하면 고객이 없을 경우 자동으로 만들어져요.
2
매출 등록할 때 자동 등록시술 끝나고 매출 등록하면 고객이 자동 생성돼요.
3
고객 탭에서 직접 등록"새 고객 등록" → 이름, 전화번호, 메모만 넣으면 끝.
07

다국어 상담

처음 오는 고객이나 외국인 고객에게만 필요한 기능이에요. 단골은 상담 없이 바로 시술 + 매출 등록하면 돼요.

한국어, 영어, 중국어, 일본어 4개 언어를 지원해요.

상담 단계

1
고객 정보이름, 연락처 입력
2
시술 기본손/발, 범위 선택
3
디자인 선택원컬러, 그라데이션, 아트 등
4
캔버스필요 시 네일 디자인을 직접 그리기
5
특성 확인표현, 세부 사항 선택
✓
최종 확인요약 보고 저장

사전상담 (미리정하기) 링크 활용

고객이 방문 전에 본인 모국어로 직접 상담 내용을 작성할 수 있어요.

  • 예약 등록 후 "상담 링크 보내기" → QR 코드 또는 링크 복사로 전달
  • 고객이 모바일에서 언어를 선택하고 디자인, 네일 상태, 참고 사진을 직접 입력
  • 완료 시 원장님 앱의 예약 카드에 사전상담 결과가 표시돼요
일정 앞당기기 팁: 사전상담 링크를 미리 보내두면 방문 시 상담 시간이 크게 줄어요.
08

결제 흐름

결제 처리 방식이 두 가지 있어요:

매출 등록 (퀵세일)

금액 넣고 등록하면 바로 결제 완료.

시술 기록도 자동 생성돼요.

→ 바로 오늘 매출에 반영

상담 후 결제

상담 저장 → 시술 기록 생성 (아직 결제 전).

나중에 "결제 완료" 버튼을 눌러야 매출에 잡혀요.

→ 결제 완료 누를 때 반영
"결제 완료"를 누르기 전까지는 시술 기록에 "결제 전" 뱃지가 보여요. 누르면 사라지고 매출에 잡혀요.
09

디자이너 전환

여러 디자이너가 한 기기를 같이 쓸 때 사용해요. 넷플릭스 프로필 전환처럼 작동해요.

1
상단 프로필 아이콘 터치화면 왼쪽 위에 동그란 프로필 아이콘이 있어요.
2
디자이너 선택등록된 디자이너 목록에서 내 이름 터치.
3
PIN 4자리 입력숫자 4자리 입력하면 자동으로 전환돼요.
PIN 변경은 설정 → 매장 정보 → 디자이너 관리에서 할 수 있어요. 기본 PIN은 1234예요.
10

온보딩 — 매장 설정 가이드

처음 BDX를 시작할 때 매장 기본 정보를 설정하는 단계예요. 한 번만 하면 돼요.

1
매장 이름과 기본 정보 입력매장 이름, 전화번호, 영업시간을 설정해요.
2
서비스 카테고리별 가격 설정심플 / 프렌치 / 자석 / 아트 각 카테고리의 기본 가격을 입력해요.
3
디자이너(선생님) 등록함께 일하는 디자이너 이름과 PIN 4자리를 등록해요.
4
포트폴리오 사진 업로드대표 시술 사진을 미리 올려두면 상담 때 바로 보여줄 수 있어요.
5
홈 대시보드로 이동설정 완료 후 앱을 본격적으로 사용할 수 있어요.
경로: 최초 로그인 → 설정 화면에서 진행
11

현장모드 — 빠른 시술 관리

고객이 왔을 때 예약 카드에서 바로 시술을 시작하고 정산까지 이어가는 흐름이에요.

1
시술 시작예약 카드에서 "시술 시작" 탭
2
디자인 카테고리 선택심플 / 프렌치 / 자석 / 아트 중 선택
3
사진 선택포트폴리오에서 시술 사진 선택 또는 새로 촬영
4
시술 진행시술 타이머 + 추가 옵션 관리
5
정산최종 가격 확인 → 결제수단 선택
✓
Wrap-up시술 사진 저장 + 고객 연결 + 공유카드 생성
경로: 홈 → 오늘 예약 카드 → 시술 시작
12

미리정하기 — 사전 상담 링크

고객이 방문 전에 모바일로 직접 상담 내용을 작성할 수 있는 기능이에요. 방문 시 상담 시간을 크게 줄여줘요.

1
링크 전달예약 등록 후 "상담 링크 보내기" → QR 코드 또는 링크 복사로 고객에게 전달
2
고객이 직접 작성고객이 모바일에서 언어 선택 후 디자인, 네일 상태, 참고 사진을 직접 입력
3
원장님 앱에 알림 표시고객이 완료하면 예약 카드에 사전상담 결과 뱃지가 생겨요
4
사전상담 결과 확인예약 카드 터치 → 사전상담 내용 상세 보기
지원 언어: 한국어, 영어, 중국어, 일본어 — 고객이 스스로 언어를 선택해 작성해요.
경로: 기록 탭 → 예약 카드 → 링크 보내기
13

공유카드 — 시술 결과 공유

시술 사진을 SNS에 올리기 좋은 카드 형태로 만들어줘요. 매장 홍보에 활용하세요.

1
시술 사진 선택1~3장을 골라요
2
비율 선택9:16 (스토리용) 또는 3:4 (피드용)
3
자동 구성매장 이름 + 디자인 정보가 자동으로 카드에 포함돼요
4
이미지 다운로드이미지로 저장 후 인스타그램, 카카오 등 SNS에 공유
경로: Wrap-up → 공유카드 만들기 / 기록 상세 → 공유카드
14

포트폴리오 — 시술 작품 관리

그동안 한 시술 사진을 한 곳에 모아두는 공간이에요. 상담 때 포트폴리오를 보여주면 고객이 더 쉽게 디자인을 고를 수 있어요.

자동 수집
Wrap-up에서 사진을 저장하면 자동으로 포트폴리오에 추가돼요.
수동 업로드
기존 사진을 직접 올릴 수도 있어요.
카테고리 태그
심플 / 프렌치 / 자석 / 아트로 분류해 찾기 쉽게 관리해요.
공개/비공개
외부에 보여줄 사진만 공개로 설정할 수 있어요.
경로: 설정 → 포트폴리오 탭
15

골든타임 — 28일 재방문 알림

네일은 보통 3~4주 후 다시 받아요. BDX는 마지막 시술 후 28일이 지나면 자동으로 알려줘요.

1
